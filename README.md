# Ecommerce-revenue-analytics
End-to-end E-Commerce Revenue &amp; Customer Analytics using Python, SQL, Power BI and Basic Excel.
📊 E-Commerce Revenue & Customer Analytics

This project analyzes transactional ecommerce data (2010–2011) to identify revenue growth drivers and customer behavior patterns using Python, SQL, and Power BI.

🔧 Tools Used

• Python (Pandas)

• SQL (MySQL)

• Power BI

• DAX

📈 Key Analysis

• Monthly Revenue Trends

• Order Volume & AOV Analysis

• RFM Customer Segmentation

• Cohort-style Retention Analysis

• Revenue Contribution by Customer Segment

🚀 Key Insights

• Revenue growth was volume-driven

• November 2011 was peak month

• ~44% revenue came from Champions segment (concentration risk)
